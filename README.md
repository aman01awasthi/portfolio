# Amandeep Awasthi — Portfolio

Personal portfolio website built with React and Tailwind CSS.

## 🚀 Live Demo
https://portfolio-nine-ebon-kgavdvd9m3.vercel.app/

## 🛠️ Built With
- React.js
- Tailwind CSS v3
- Vite

## 📂 Sections
- Hero — Introduction and CTA
- About — Background and experience
- Skills — Tech stack
- Projects — Live projects with links
- Contact — Email and LinkedIn

## 🏃 Run Locally
npm install
npm run dev
